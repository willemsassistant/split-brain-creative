# Split Brain Creative

A Next.js + Tailwind MVP for the Split Brain Creative studio website.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

This is the initial production-oriented MVP:
- Home
- Work
- About
- Contact

The visual direction aims for dark, rich, editorial, and human rather than generic startup or corporate aesthetics.
